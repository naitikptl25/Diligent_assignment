# E‑Commerce Database Project (Cursor AI + SQLite)

This project builds a simple **E‑commerce database** using **SQLite** and Python.  
It includes:

- Auto‑generated mock data using Python (`faker`, `pandas`)
- SQL tables for customers, products, orders, order items
- SQL queries to join data and produce a final combined report
- Exporting the final SQL result into a CSV file
